# Crop Recommendation System
A machine learning based system that recommends the most suitable crop for a given set of soil and environmental conditions, helping farmers make data-driven planting decisions.
The system takes inputs like nitrogen, phosphorus, and potassium levels, temperature, humidity, pH, and rainfall, and predicts the best-fit crop using three classification models : Random Forest, K-Nearest Neighbors (KNN), and Support Vector Machine (SVM) , comparing their performance to select the most accurate approach.
Tech Stack:
Python
scikit-learn
Pandas, NumPy
Jupyter / Google Colab
